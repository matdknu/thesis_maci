Análisis textual longitudinal de la contestación discursiva en las elecciones presidenciales de 2025 en Chile

Autor/a

Matías Deneken

Este capítulo presenta el bloque de modelado con foco en la pregunta central de la tesis: cómo se organiza y predice la polarización política en conversación digital. El criterio editorial es explícito: todo se reporta en PNG y CSV, sin recursos HTML interactivos, para asegurar trazabilidad y consistencia entre HTML y PDF.

## 1) Desempeño de modelos supervisados

```{r tab-sentimiento}
path_sent <- file.path(base_out, "resultados_sentimiento.csv")
if (file.exists(path_sent)) {
  s <- read.csv(path_sent, stringsAsFactors = FALSE)
  num_cols <- intersect(c("accuracy", "f1_macro"), names(s))
  for (cn in num_cols) s[[cn]] <- round(s[[cn]], 3)
  knitr::kable(s, caption = "Resultados de clasificación de sentimiento (test).", booktabs = TRUE)
}
```

```{r fig-cm-sent, fig.cap="Matriz de confusión para clasificación de sentimiento (PNG)."}
if (file.exists(file.path(base_out, "cm_sentimiento.png"))) {
  knitr::include_graphics(file.path(base_out, "cm_sentimiento.png"))
}
```

```{r tab-polar}
path_pol <- file.path(base_out, "resultados_polarizacion.csv")
if (file.exists(path_pol)) {
  p <- read.csv(path_pol, stringsAsFactors = FALSE)
  for (cn in intersect(c("MAE", "RMSE", "R2"), names(p))) p[[cn]] <- round(p[[cn]], 4)
  knitr::kable(p, caption = "Resultados de regresión para polarización (test).", booktabs = TRUE)
}
```

```{r tab-disc-vs-text}
path_cmp <- file.path(base_out, "comparacion_regresion_discursiva_vs_texto.csv")
```

## 2) Variables que más explican polarización

```{r tab-importance}
path_imp <- file.path(base_out, "feature_importance_polarizacion.csv")
if (file.exists(path_imp)) {
  imp <- read.csv(path_imp, stringsAsFactors = FALSE)
  imp <- head(imp, 20)
  if ("importance" %in% names(imp)) imp$importance <- round(imp$importance, 4)
  knitr::kable(imp, caption = "Top 20 rasgos de mayor importancia para predecir polarización.", booktabs = TRUE)
```
